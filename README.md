# Django Multiuser Blog

## Description
Django Multiuser Blog est une application web de blog permettant à plusieurs utilisateurs de gérer leur propre contenu. Chaque utilisateur peut s'inscrire, se connecter et créer des articles. Le système propose également des commentaires pour favoriser l'interaction entre auteurs et lecteurs.

## Fonctionnalités
- Inscription et authentification des utilisateurs (système Django intégré).
- Création, édition et suppression d’articles.
- Gestion des catégories et étiquettes.
- Système de commentaires.
- Interface d’administration personnalisée.

## Technologies
- Django
- SQLite (par défaut, configurable avec PostgreSQL, MySQL, etc.)
- HTML5, CSS3, JavaScript (pour l’interface)
- Bootstrap (pour le design)
